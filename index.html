Introduction

mygpoclient-java is my attempt to port the Python mygpoclient to Java. I've tried to stay as close to the original Python source as possible where I could so that developers that are comfortable with the Python version will be at home.

Getting Started

This project uses Maven to manage dependencies and the build process.

  1. Clone the git repository git clone https://github.com/Dragontek/mygpoclient-java.git
  2. Create Eclipse project with Maven mvn eclipse:eclipse
  3. Create client.properties file in the root folder of the project.
  4. Run tests mvn test
